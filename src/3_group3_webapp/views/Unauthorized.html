🔒 ไม่มีสิทธิ์เข้าถึง

บัญชี Google ของคุณไม่อยู่ในรายชื่อผู้ใช้ที่ได้รับอนุญาต สำหรับ LMDS Dashboard

บัญชีที่ล็อกอินอยู่:

ไม่สามารถอ่านได้

📋 วิธีขอสิทธิ์การเข้าถึง:

  1. ติดต่อผู้ดูแลระบบ LMDS
  2. แจ้งอีเมล Google Account ของคุณ
  3. แจ้งให้เพิ่มใน Script Property DASHBOARD_USERS
  4. รอผู้ดูแลยืนยัน แล้วรีโหลดหน้านี้
